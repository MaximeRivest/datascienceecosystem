Polars Bank-Chunked Shim

This page tests the earlier shim idea directly. Instead of first materializing one contiguous arena, it fills a bank-like segmented layout inside Wasm memory and asks Polars to consume those fragments as chunked columns. It compares three paths: generated baseline, contiguous ownership-transfer arena, and chunked segmented shim.
