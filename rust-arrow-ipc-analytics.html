MVT 3e: Rust Arrow IPC Parsing over the Zero-Copy Shard Bank

This prototype routes Arrow IPC batches into a shared WebAssembly.Memory shard bank, then uses 4 Rust/Wasm workers with the Rust Arrow IPC reader to parse those batches directly from the imported shard memory and compute real aggregates.
