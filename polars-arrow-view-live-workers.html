Polars Arrow-View Live Workers

This is the closest currently implementable direct follow-on to the live Arrow-style shard-bank view proof. It uses the same shared bank + manifest, runs a fresh-worker live-view aggregate, and in parallel runs a fresh-worker Polars-from-bank path where workers materialize Polars-owned arenas directly from the live bank partitions they were assigned.

Important nuance: with the current browser Polars Wasm packaging, this page does not yet make Polars read external shared bank memory directly. Instead, it proves that Polars can consume data directly from the real live bank manifest in fresh workers and that both the live-view path and the Polars path track shared-bank mutations exactly.
