Polars Partition-Arena Workers

This is the next-step prototype after the monolithic-memory proof. Instead of trying to force one giant memory into Polars, it splits the workload across workers and gives each worker a contiguous Polars-friendly partition arena. The page compares a distributed Polars-generated baseline against a distributed ownership-transfer arena path.

Ready. Run the distributed partition-arena proof.
Partition-arena setup

Important caveat: this page is not direct shard-bank integration yet. It is the recommended next-step proof for a bank -> partition arena -> Polars architecture. Each worker owns one contiguous partition arena and Polars adopts those buffers by ownership transfer.
