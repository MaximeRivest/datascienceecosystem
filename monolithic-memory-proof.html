Monolithic Memory Proof

This page tests whether a single giant memory object is a viable substitute for the proven sharded bank. It runs two separate probes: one for a monolithic SharedArrayBuffer and one for a monolithic shared WebAssembly.Memory. The second probe is the one that matters most for Rust/Wasm analytics engines.
